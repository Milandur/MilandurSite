# Сито
У нас на сервере вы можете превратить песок в кварц, например, для случаев, когда доступа в ад нет.\

## Создание
* Для создания сита необходимо, совместить 5 дубовых досок и 1 железные прутья
![<Крафт>](/sito_craft.png)
## Использование
* Нужно взять песок в левую руку
* Нажать ПКМ по воздуху
* Повторить еще 3 раза для песка
